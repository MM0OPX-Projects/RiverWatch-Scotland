# RiverWatch Scotland

Your guide to Scotlands river fishing conditions.

## Files

- `index.html` - browser-based HTML version of RiverWatch Scotland.
- `apk/RiverWatch-Scotland-v0.1-debug.apk` - Android debug APK build.

## Data Sources

RiverWatch Scotland uses SEPA Water Levels, the SEPA Time Series API, and Open-Meteo forecast data.
